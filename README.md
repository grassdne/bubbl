# Bubbl
![logo](images/logo.gif)

A WIP set of animations, simulations, and games with a common theme.

### Dependencies
SDL2, LuaJIT, OpenGL 3.3

### Fedora Example
```bash
sudo dnf install SDL2-devel glew-devel
./build.lua
./bubbl
```
