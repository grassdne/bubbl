# Bubbl
An easy-to-use interface for creating simulations and animations with a peculiar set of graphics primitives.

### Dependencies
SDL2, LuaJIT, GLEW (OpenGL 3.3)

### Fedora Example
```bash
sudo dnf install SDL2-devel glew-devel luajit-devel
./build.sh
./bubbl --help
```
